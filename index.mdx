## What is NoodleMapper?

NoodleMapper is a plugin for [ChroMapper](https://github.com/Caeden117/ChroMapper), the community Beat Saber map editor. It extends ChroMapper with three major capabilities:

**Map range annotation** — Draw named, color-coded bars along the editor timeline to mark sections of your map (intro, verse, chorus, drop, etc.). Ranges are stored alongside your difficulty files and are purely a mapper's tool — they don't affect gameplay.

**Modmap management** — Attach a `.modmap` file to any difficulty to store NoodleExtensions effect spans. A modmap defines time-based spans that target specific notes (by hand type, and more) and apply custom track assignments or other processors to them.

**Git version control** — A Git panel lives directly inside ChroMapper. Initialise a repo, commit your work, manage remotes, and push to GitHub without ever leaving the editor. If you have the GitHub CLI (`gh`) installed, NoodleMapper can create a private GitHub repository for you in one click.

## Key features

<CardGroup cols={2}>
  <Card title="Timeline range bars" icon="bars-progress">
    Annotate any beat span with a name and color. Ranges sync colors across matching names automatically.
  </Card>
  <Card title="Modmap files" icon="file-code">
    Create and assign `.modmap` files per difficulty. Each modmap stores effect spans with filters and processors.
  </Card>
  <Card title="Noodle span filters" icon="filter">
    Target left-hand or right-hand notes within a span using the `HandTypeFilter`. The filter system is extensible via `INoodleFilter`.
  </Card>
  <Card title="Git panel" icon="git-alt">
    Init, commit, fetch, pull, push, and manage remotes — all from inside the Song Edit Menu.
  </Card>
  <Card title="GitHub clone window" icon="github">
    Browse your GitHub map repositories and clone them directly into CustomWIPLevels from the Song Select screen.
  </Card>
  <Card title="Per-difficulty data files" icon="folder-tree">
    Each difficulty gets its own `.map` file named automatically after the difficulty + characteristic (e.g., `ExpertPlusStandard.map`).
  </Card>
</CardGroup>

## How it fits together

```
YourSong/
├── Info.dat                      ← Beat Saber song info
├── ExpertPlusStandard.dat        ← Standard difficulty
└── NoodleMapper/
    ├── ExpertPlusStandard.map    ← NoodleMapper data (ranges + modmap link)
    └── MyEffects.modmap          ← Modmap effect spans
```

The `.map` file is referenced from the difficulty's `CustomData` under the `"_NoodleMapperMap"` key. The `.modmap` file is referenced from the `.map` file under `"modMap"`. Both files are JSON and live in a `NoodleMapper/` folder inside your song directory.

<Note>
NoodleMapper only activates on a difficulty when you explicitly enable it. Difficulties without the `_NoodleMapperMap` custom data key are untouched.
</Note>
